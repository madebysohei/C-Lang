標準入力（ほとんどはキーボード）から値を読み取るために使う関数 <stdio.h> に含まれている。

書式
==scanf("フォーマット", &変数);==

用例
int age;
printf("年齢を入力してください：");
==scanf("%d", &age); ==


| 指定子   | 意味          | 例                     |
| ----- | ----------- | --------------------- |
| `%d`  | int（整数）     | `scanf("%d", &num);`  |
| `%f`  | float（小数）   | `scanf("%f", &pi);`   |
| `%lf` | double（倍精度） | `scanf("%lf", &val);` |
| `%c`  | 1文字         | `scanf("%c", &ch);`   |
| `%s`  | 文字列         | `scanf("%s", str);`   |
